# Lab.1

試寫一程式，讀取指定的檔案，並將每一行中遇到的

* 英文字母依據大小寫分別轉換為 A 或 a
* 數字轉換為 1

後印出於螢幕上，最後計算並印出轉換後的 A、a 與 1 的數量。

## Example

```
<?xml version='1.0' encoding='UTF-8'?>
<RDoC>
<TEXT><![CDATA[
```

會轉換成

```
<?aaa aaaaaaa='1.1' aaaaaaaa='AAA-1'?>
<AAaA>
<AAAA><![AAAA[

```
